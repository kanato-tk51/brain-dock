# 個人ライフOS 段階的ロードマップ

## 優先順位の基準
1. 価値が高い（忘れっぽさに直撃）
2. 依存が少ない（単独で動く）
3. 失敗しても復旧しやすい（データロストしない）
4. 継続しやすい（入力が軽い）

## Phase 0: 基盤と安全の最小セット（2〜3日）
### 達成できること
- DB正本の最小スキーマが動く
- 1行入力を受けるInboxが動く
- 最低限の機密スキャナが動く

### 入力方法
- CLI: `brain capture "テキスト" --domain work --sensitivity normal`
- 手入力のみ（最初は入力チャネルを絞る）

### 参照できること
- キーワード検索
- 日付フィルタ
- 直近ログ一覧

### 次フェーズへの繋がり
- 後続の自動整理と意味検索の受け皿になる

## Phase 1: 継続入力 + 即時想起MVP（1〜2週間）
### 達成できること
- 日次ログ/学び/実績/TODOを同じ入り口で蓄積
- 「会話で使える実績カード」を数秒で引ける
- 週次サマリを自動生成できる

### 入力方法
- CLIクイックキャプチャ
- スマホショートカット（Webhook経由でInbox投入）
- URL保存（最低限のブラウザ送信）

### 参照できること
- 自然文検索（例: 「去年の障害対応で学んだこと」）
- 人物/プロジェクト/期間で絞り込み
- 会話用3行要約

### 次フェーズへの繋がり
- 自動タグ付け・エンティティ抽出を入れて整理コストを減らせる

## Phase 2: 自動整理エンジン（2〜4週間）
### 達成できること
- タグ付け、人物/場所/プロジェクト抽出
- 表記ゆれ統合（エイリアス辞書）
- TODO/決定ログの抽出

### 入力方法
- Phase 1同様（入力を増やさず自動化を追加）

### 参照できること
- 人物ページ・プロジェクトページの自動更新
- 未整理Inboxの分類提案

### 次フェーズへの繋がり
- 検索精度向上と会話アシストの検索品質向上

## Phase 3: 検索・参照の強化（2〜3週間）
### 達成できること
- 全文 + 類似検索のハイブリッド検索
- 初出検索、差分振り返り、時間軸ビュー
- 閲覧中コンテンツへの関連ログ提示

### 入力方法
- 既存入力を継続（追加入力不要）

### 参照できること
- 「断片的記憶」からの再発見
- 先月との差分や再訪候補

### 次フェーズへの繋がり
- 会話中提案の根拠提示が可能になる

## Phase 4: 会話アシストBeta（4〜8週間）
### 達成できること
- リアルタイム文字起こし + 論点抽出
- 「今言うべき要点」「関連経験」「短い返答案」を提示
- 会話後に決定事項/宿題を自動記録

### 入力方法
- 音声ストリーム（保存ポリシーで制御）
- 会話後の確認コマンドで確定保存

### 参照できること
- 相手別の話し方ヒント
- 実績引き出しカード

### 次フェーズへの繋がり
- 自動運転（毎朝ブリーフィング等）に発展

## Phase 5: 自動運転と個人エージェント化（継続）
### 達成できること
- 今日の3つ提案、夜の3行振り返り
- 価値観整合チェック、意思決定追跡
- 仕事力/人間関係/習慣の半自動オペレーション

### 入力方法
- 低頻度レビュー入力のみ

### 参照できること
- 今の自分カード、長期傾向、リスクレジスター

### 次フェーズへの繋がり
- 分析機能を安全配慮下で追加

## Phase 6: 高度分析（必要最小限）
### 達成できること
- 成長要因推定、意思決定癖分析
- 偏り可視化（センシティブは匿名化/集計化）

### 入力方法
- 自動収集データを再利用

### 参照できること
- 介入効果（何が効いたか）
- 次の改善アクション
